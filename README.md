## Running Locally
```sh
$ git clone git@github.com:danner/tetra_image_gallery.git
$ cd tetra_image_gallery
$ npm install
$ npm start
```

Your app should now be running on [localhost:5000](http://localhost:5000/).

## Deploying to Heroku

```
$ heroku create
$ git push heroku master
$ heroku open
```
